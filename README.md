# Etch-A-Sketch
A classic, grid based drawing program, following a few guidelines, This was recently visually updated with a few tweaks to the code, there are issues with it, but I wanted to keep it true to my beginnings.

#### The Good
- Grid Resizing.
- Keyboard Shortcuts.

#### The Bad
- Clunky To Use.
- Desktop Only.
